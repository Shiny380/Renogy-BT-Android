# Renogy Bluetooth Android

Android app to replace the official Renogy DC Home app to avoid creating an account and needing internet connection.

**CAUTION:** I didn't write a single line of code for this, it's entirely AI (mostly Gemini Code)

Currently only supports:
- Bluetooth Battery
- DC Charger
